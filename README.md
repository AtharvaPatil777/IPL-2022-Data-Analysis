# 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the IPL 2022 dataset to uncover insights about team performance, player statistics, venues, toss decisions, and winning patterns.

The project demonstrates real-world data cleaning, visualization, and analytical thinking using Python.

---

# 🎯 Business Objectives

- Analyze IPL 2022 match data
- Identify top-performing teams
- Identify top-performing players
- Analyze venue-wise performance
- Study toss impact on winning
- Find highest winning margins
- Answer business-oriented analytical questions
