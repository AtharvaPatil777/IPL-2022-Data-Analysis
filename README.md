🏏 IPL 2022 Data Analysis using Python
📌 Project Overview

This project performs an exploratory data analysis (EDA) on the IPL 2022 dataset to uncover insights about team performance, player statistics, match outcomes, venues, and winning patterns. Using Python and popular data analysis libraries, the project demonstrates data cleaning, visualization, and business-oriented analysis.

🎯 Objectives
Analyze IPL 2022 match data.
Identify top-performing teams and players.
Explore venue-wise performance.
Understand toss impact on match outcomes.
Visualize key trends using charts.
Answer business-driven analytical questions.
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Plotly (if used)
Jupyter Notebook
📊 Key Insights
Team with the highest number of wins.
